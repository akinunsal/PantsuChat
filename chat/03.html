Easy Notification plugin: Demo 3

Cookie based notifications. Notification will only show on first page visit. If you don't see the notification above, you have already visited this page.

Usage

$(function(){		   
    $.easyNotification({
        cookieEnable: true,
        text: 'Cookie enabled notification. This is the only time you will see the notification!'
    }); 
});
